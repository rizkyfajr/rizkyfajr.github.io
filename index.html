e-KFPB

Mengembangkan dan memelihara aplikasi internal untuk Kimia Farma Plant Banjaran (KFPB) yang digunakan untuk mengelola berbagai data operasional perusahaan. Saya bertanggung jawab pada beberapa modul kunci.

  • SPPTEK: Modul untuk permintaan pembelian barang teknik.
  • QMS: Modul Quality Management System untuk pengendalian dokumen, change control, dan penyimpangan.

Teknologi: PHP Native, SQL, jQuery, Bootstrap.

eSDM

Membangun aplikasi eSDM untuk manajemen sumber daya manusia, dengan fokus utama pada sistem absensi karyawan di Kimia Farma. Aplikasi ini memiliki dua level pengguna (Admin dan Pegawai).

  • Fitur Utama: Pencatatan absensi, manajemen data pegawai, rekapitulasi laporan.
  • Integrasi API: Mengembangkan API untuk sinkronisasi data pegawai dan absensi ke sistem pusat Kimia Farma.

Teknologi: PHP, SQL, jQuery, REST API.
